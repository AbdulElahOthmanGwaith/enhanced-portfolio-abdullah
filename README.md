# ملف التعريف الشخصي المطور - عبدالاله عثمان غويث

هذا المشروع هو نسخة مطورة ومحسنة من الملف الشخصي (Portfolio) للمطور عبدالاله عثمان غويث. تم تحسين الأداء، تجربة المستخدم، وتهيئة محركات البحث (SEO).

## التحسينات المضافة:
- **تحسين SEO**: إضافة وسوم وصفية متقدمة وكلمات مفتاحية لتحسين الظهور في محركات البحث.
- **تأثيرات بصرية**: إضافة تأثيرات "Reveal on Scroll" لجعل تصفح الموقع أكثر تفاعلية.
- **تحسين الأداء**: تنظيم ملفات CSS و JS وفصل التحسينات الجديدة في ملفات مستقلة.
- **تجربة المستخدم (UX)**: تحسين استجابة الأزرار وبطاقات المشاريع.

## التقنيات المستخدمة:
- HTML5
- CSS3 (مع متغيرات CSS وتأثيرات متقدمة)
- JavaScript (Vanilla JS)

## كيفية التشغيل:
ببساطة قم بفتح ملف `index.html` في أي متصفح ويب حديث.
